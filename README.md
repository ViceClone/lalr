# A Simple LALR Parser
main.cpp contains the demo program for the lexer.
main2.cpp contains the demo program for the parser.
## Build the project
### Lexer
```
make lexer
```
### Parser
```
make parser
```
or simply
```
make
```
## Run demo
### Lexer
```
./lexer
```
### Parser
```
./parser
```
## Clean
```
make clean
```
